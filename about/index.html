我

村中身高扛把子，双眼度数 500+ ，甚爱大黑框。

会爬树 🌳 ，会锄地 🎑 ，会游泳 🏊 ，会做饭 🍚 ，会吹牛 🎤 ，会绘画🎨。
会在下雨天睡觉，也会在起雾的玻璃上画心。

本站

CMS 用的是 Hugo，服务源于 Github Pages，源代码托管 GitHub。
评论使用 Beaudar - 表达，基于 GitHub issues，甚是方便。
曾经使用过 Wildfire 🔥，UI很不错，但是要用 Firebase ，在国内很不稳定。
DNS、CDN 和 SSL 来自 Cloudflare 。
