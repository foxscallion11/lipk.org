网站升级

李鹏坤 / 2020-04-29

最近几天网站我做了很多修改。
域名从 zsdycs.cn 转到了 lipk.org，前者这几年仍将继续使用，.cn 来自阿里云，后者 .org 我票的是 Google domain。 最大的感想就是谷歌的控制台比阿里云的好看很多很多。
式样外观，最大的不同就是较窄显示区域时，菜单停留在了顶部。 菜单区域的手绘图片被我去掉了。首页后续有个非常大的改动，黑夜白天模式切换放到了菜单里，而且实现方式从 Darkmode.js （是使用 mixColor 实现的），现在改为了 disabled CSS link 的方式，这样更加可控，浏览器兼容性也更好。
